# Инструкция для загрузки файлов и входа в админку

## Что это за сайт
Это простой сервис для загрузки файлов участниками и просмотра загрузок в админке.

## Куда заходить
- **Страница входа для загрузки:** `http://localhost:3000/upload-login`
- **Страница загрузки:** `http://localhost:3000/`
- **Админка:** `http://localhost:3000/admin`

## Логин и пароль
Для входа на страницу загрузки и в админку используются **одинаковые** данные:
- Логин: **admin**
- Пароль: **password**

## Как загрузить файл (самый простой вариант)
1. Откройте страницу входа для загрузки: `http://localhost:3000/upload-login`.
2. Введите логин и пароль (см. выше) и нажмите **Войти**.
3. Вы попадете на страницу загрузки.
4. Введите **ФИО**.
5. Нажмите **Выбрать файл** и выберите файл на компьютере.
6. Нажмите **Загрузить**.
7. Дождитесь сообщения об успешной загрузке.

### Важно: если файлов несколько
**Не загружайте файлы по одному.**
Сначала соберите все файлы в **один архив (.zip)**, затем загрузите архив как один файл.

Подсказка (общие шаги):
- Выделите нужные файлы → «Сжать»/«Архивировать» → получится `.zip`.
- Загружайте получившийся архив.

## Как открыть админку
1. Откройте `http://localhost:3000/admin`.
2. Введите логин и пароль (см. выше) и нажмите **Войти**.
3. В админке вы увидите список загруженных файлов и сможете их скачать.

## Если что-то не получается
- Если при загрузке вас перекидывает на вход — значит вы не авторизованы. Зайдите на `http://localhost:3000/upload-login`.
- Проверьте, что заполнено поле **ФИО** и выбран файл.
- При проблемах со входом проверьте логин и пароль.
